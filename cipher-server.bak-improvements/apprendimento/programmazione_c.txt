# Linguaggio C – Conoscenza Base

## Introduzione
C è un linguaggio di programmazione procedurale di basso livello, creato da Dennis Ritchie negli anni '70. È alla base di molti sistemi operativi (Linux, Windows) e linguaggi moderni. Offre controllo diretto sulla memoria e sull'hardware.

---

## Struttura base di un programma

```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
```

- `#include`: direttiva del preprocessore per includere librerie
- `main()`: punto di ingresso del programma
- `return 0`: indica esecuzione completata con successo

## Operatori

- Aritmetici: +, -, *, /, % (modulo)
- Relazionali: ==, !=, <, >, <=, >=
- Logici: && (AND), || (OR), ! (NOT)
- Bitwise: &, |, ^, ~, <<, >>
- Assegnazione: =, +=, -=, *=, /=

## Controllo del flusso

```c
// if-else
if (x > 0) {
    printf("positivo");
} else if (x < 0) {
    printf("negativo");
} else {
    printf("zero");
}

// switch
switch (scelta) {
    case 1: printf("uno"); break;
    case 2: printf("due"); break;
    default: printf("altro");
}

// for
for (int i = 0; i < 10; i++) {
    printf("%d\n", i);
}

// while
while (condizione) { ... }

// do-while
do { ... } while (condizione);
```

## Puntatori

Uno dei concetti più potenti e complessi del C.

```c
int x = 10;
int *ptr = &x;   // ptr contiene l'indirizzo di x

printf("%d", *ptr);  // dereferenziazione: stampa 10
*ptr = 20;           // modifica x tramite puntatore

// Puntatore a array
int arr[] = {1, 2, 3};
int *p = arr;
printf("%d", *(p + 1));  // stampa 2
```

## Gestione della memoria (heap)

```c
#include <stdlib.h>

int *arr = malloc(10 * sizeof(int));  // alloca memoria
if (arr == NULL) { /* errore */ }

arr[0] = 42;

free(arr);  // SEMPRE liberare la memoria
arr = NULL;
```

- `malloc`: alloca memoria non inizializzata
- `calloc`: alloca e inizializza a zero
- `realloc`: ridimensiona blocco allocato
- `free`: libera la memoria

## Stringhe

In C le stringhe sono array di char terminati da '\0'.

```c
#include <string.h>

char s1[20] = "ciao";
char s2[] = "mondo";

strlen(s1);           // lunghezza
strcpy(s1, s2);       // copia
strcat(s1, s2);       // concatena
strcmp(s1, s2);       // confronta (0 se uguali)
```

## File I/O

```c
FILE *f = fopen("file.txt", "r");  // r=lettura, w=scrittura, a=append
if (f == NULL) { /* errore */ }

char riga[256];
while (fgets(riga, sizeof(riga), f)) {
    printf("%s", riga);
}

fclose(f);
```

## Librerie standard principali

- `stdio.h`: I/O (printf, scanf, fopen, ecc.)
- `stdlib.h`: memoria, conversioni (malloc, atoi, ecc.)
- `string.h`: manipolazione stringhe
- `math.h`: funzioni matematiche
- `time.h`: gestione del tempo

gcc -g -o programma main.c       # con debug info
gcc -O2 -o programma main.c      # con ottimizzazioni
