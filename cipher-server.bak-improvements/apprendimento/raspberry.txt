# Raspberry Pi – Guida Completa e Moduli

---

## Cos'è Raspberry Pi

Raspberry Pi è un single-board computer (SBC) sviluppato dalla Raspberry Pi Foundation (UK) a partire dal 2012. Economico, compatto e versatile, è usato per apprendimento, progetti IoT, server, retrogaming, domotica, robotica e molto altro. Gira principalmente su Linux (Raspberry Pi OS).

---

## Storia e modelli

### Raspberry Pi 1 (2012)
- CPU: ARM1176JZF-S 700MHz single-core
- RAM: 256MB / 512MB
- Versioni: Model A, Model B
- Prima SBC economica accessibile al grande pubblico

### Raspberry Pi 2 (2015)
- CPU: ARM Cortex-A7 900MHz quad-core
- RAM: 1GB
- Prima versione con USB 2.0 x4 e Ethernet

### Raspberry Pi 3 (2016)
- CPU: ARM Cortex-A53 1.2GHz quad-core 64-bit
- RAM: 1GB
- WiFi 802.11n + Bluetooth 4.1 integrati (prima volta)
- Versioni: Model B, Model B+

### Raspberry Pi 4 (2019)
- CPU: ARM Cortex-A72 1.5GHz quad-core 64-bit
- RAM: 1GB / 2GB / 4GB / 8GB
- USB 3.0, USB-C power, dual micro-HDMI (4K)
- WiFi 802.11ac + Bluetooth 5.0
- Gigabit Ethernet
- Il modello più diffuso per uso general purpose

### Raspberry Pi 5 (2023) — quello di Szymon
- CPU: ARM Cortex-A76 2.4GHz quad-core 64-bit
- RAM: 4GB / 8GB
- PCIe 2.0 (primo Pi con PCIe)
- USB 3.0 x2, USB 2.0 x2
- WiFi 802.11ac + Bluetooth 5.0
- Gigabit Ethernet
- Nuovo connettore per display e camera (MIPI)
- Chip RP1 dedicato agli I/O
- Significativo salto prestazionale rispetto al Pi 4

### Raspberry Pi Zero (2015)
- CPU: ARM1176JZF-S 1GHz single-core
- RAM: 512MB
- Formato ultra-compatto, economico (~5$)
- Versioni: Zero, Zero W (WiFi/BT), Zero 2 W (quad-core)

### Raspberry Pi Pico (2021)
- Microcontrollore (non SBC) basato su RP2040
- Dual-core ARM Cortex-M0+ 133MHz
- 264KB SRAM, 2MB Flash
- Programmato in MicroPython o C/C++
- Simile ad Arduino, non gira Linux

### Compute Module (CM)
- Versioni per uso industriale/embedded
- Senza porte esterne, montato su carrier board custom
- CM1, CM3, CM4 (con WiFi opzionale, eMMC)

## Raspberry Pi OS

Sistema operativo ufficiale basato su Debian.
- **Desktop**: con interfaccia grafica (PIXEL/LXDE)
- **Lite**: headless, solo terminale, più leggera
- **Full**: desktop + software aggiuntivo preinstallato

### Altri OS supportati
- Ubuntu Server / Ubuntu Desktop
- Kali Linux (arm64)
- RetroPie (emulazione retrogaming)
- LibreELEC / OSMC (media center)
- Home Assistant OS (domotica)
- DietPi (ultra-leggero)
- Armbian

## Setup iniziale

```bash
# Scrittura immagine su SD
# Usare Raspberry Pi Imager (ufficiale)
# oppure:
dd if=raspios.img of=/dev/sdX bs=4M

# Prima configurazione
sudo raspi-config
# Opzioni: locale, timezone, SSH, WiFi, overclock, ecc.

# Aggiornamento sistema
sudo apt update && sudo apt upgrade -y

# Abilitare SSH
sudo systemctl enable ssh
sudo systemctl start ssh

# Trovare IP
hostname -I
ip addr show
```

## GPIO (General Purpose Input/Output)

Il cuore dell'interazione hardware del Pi.

### Pinout Pi 4/5 (40 pin header)
```
Pin  1: 3.3V           Pin  2: 5V
Pin  3: GPIO2 (SDA1)   Pin  4: 5V
Pin  5: GPIO3 (SCL1)   Pin  6: GND
Pin  7: GPIO4          Pin  8: GPIO14 (TXD)
Pin  9: GND            Pin 10: GPIO15 (RXD)
Pin 11: GPIO17         Pin 12: GPIO18 (PWM0)
Pin 13: GPIO27         Pin 14: GND
Pin 15: GPIO22         Pin 16: GPIO23
Pin 17: 3.3V           Pin 18: GPIO24
Pin 19: GPIO10 (MOSI)  Pin 20: GND
Pin 21: GPIO9  (MISO)  Pin 22: GPIO25
Pin 23: GPIO11 (SCLK)  Pin 24: GPIO8 (CE0)
Pin 25: GND            Pin 26: GPIO7 (CE1)
Pin 27: GPIO0 (ID_SD)  Pin 28: GPIO1 (ID_SC)
Pin 29: GPIO5          Pin 30: GND
Pin 31: GPIO6          Pin 32: GPIO12 (PWM0)
Pin 33: GPIO13 (PWM1)  Pin 34: GND
Pin 35: GPIO19 (MISO)  Pin 36: GPIO16
Pin 37: GPIO26         Pin 38: GPIO20 (MOSI)
Pin 39: GND            Pin 40: GPIO21 (SCLK)
```

### ATTENZIONE GPIO
- Tensione logica: **3.3V** (NON 5V — si brucia!)
- Corrente massima per pin: 16mA
- Corrente totale GPIO: 50mA
- Usare level shifter per collegare a dispositivi 5V

### Programmazione GPIO con Python (RPi.GPIO)
```python
import RPi.GPIO as GPIO
import time

GPIO.setmode(GPIO.BCM)      # numerazione BCM
GPIO.setup(17, GPIO.OUT)    # pin 17 come output
GPIO.setup(4, GPIO.IN, pull_up_down=GPIO.PUD_UP)  # input con pull-up

GPIO.output(17, GPIO.HIGH)  # accendi
time.sleep(1)
GPIO.output(17, GPIO.LOW)   # spegni

stato = GPIO.input(4)       # leggi input

GPIO.cleanup()              # rilascia GPIO
```

### GPIO con gpiozero (più semplice)
```python
from gpiozero import LED, Button, DistanceSensor
from time import sleep

led = LED(17)
led.on()
sleep(1)
led.off()
led.blink(on_time=0.5, off_time=0.5)

button = Button(4)
button.wait_for_press()
print("Premuto!")
```

## Interfacce disponibili

### UART
```bash
# Abilita in raspi-config → Interface Options → Serial
# /dev/ttyAMA0 (Pi 3/4/5) o /dev/ttyS0

import serial
ser = serial.Serial('/dev/ttyS0', 9600)
ser.write(b'Hello')
data = ser.readline()
```

### I²C
```bash
sudo raspi-config → Interface Options → I2C → Enable
# Scansiona dispositivi
sudo i2cdetect -y 1
```
```python
import smbus2
bus = smbus2.SMBus(1)
bus.write_byte_data(0x48, 0x00, 0xFF)
data = bus.read_byte_data(0x48, 0x00)
```

### SPI
```bash
sudo raspi-config → Interface Options → SPI → Enable
# /dev/spidev0.0 e /dev/spidev0.1
```
```python
import spidev
spi = spidev.SpiDev()
spi.open(0, 0)
spi.max_speed_hz = 1000000
response = spi.xfer2([0x01, 0x02])
```

### Camera
```bash
sudo raspi-config → Interface Options → Camera → Enable
# Nuova libreria (Pi 4/5)
libcamera-still -o foto.jpg
libcamera-vid -o video.h264 -t 10000
```
```python
from picamera2 import Picamera2
picam2 = Picamera2()
picam2.start()
picam2.capture_file("foto.jpg")
```

## HAT (Hardware Attached on Top)

Schede di espansione che si montano direttamente sui 40 pin GPIO.

### Sense HAT
- Accelerometro, giroscopio, magnetometro
- Sensore pressione/temperatura/umidità
- Display LED 8x8 RGB
- Joystick
- Usato sulla ISS (International Space Station)
```python
from sense_hat import SenseHat
sense = SenseHat()
sense.show_message("Ciao!")
temp = sense.get_temperature()
```

### Camera Module
- V1: 5MP OmniVision OV5647
- V2: 8MP Sony IMX219
- V3: 12MP Sony IMX708 (con autofocus)
- HQ Camera: 12.3MP Sony IMX477 (con ottiche intercambiabili)
- Collega tramite ribbon cable CSI

### Display ufficiali
- **7" Touch Display**: 800x480, touch capacitivo
- **Display 2**: 720x1280 IPS

### Pimoroni HAT Pack
- **Blinkt!**: 8 LED APA102
- **Automation HAT**: relè, ADC, uscite digitali
- **EnviroHAT**: sensori ambientali
- **Fan SHIM**: ventola per raffreddamento
- **NVMe Base**: SSD M.2 via PCIe (per Pi 5)

### Waveshare HAT
- Display E-Ink vari formati
- Motor Driver HAT
- Servo Driver HAT (PCA9685)
- OLED HAT
- LCD HAT con joystick
- CAN Bus HAT
- RS485 HAT
- GPS HAT

### Altri HAT popolari
- **PoE HAT**: Power over Ethernet
- **UPS HAT**: alimentazione da batteria con ricarica
- **RTC HAT**: Real Time Clock (DS3231)
- **Audio HAT**: HiFiBerry, IQaudio (audio di qualità)
- **LoRa HAT**: comunicazione long range
- **GSM/4G HAT**: connettività cellulare
- **RFID HAT**: lettura tag NFC/RFID

## Moduli e componenti comuni per Pi

### Comunicazione
- **HC-05/HC-06**: Bluetooth seriale
- **ESP8266/ESP32**: WiFi aggiuntivo / bridge
- **NRF24L01**: radio 2.4GHz
- **LoRa SX1276**: comunicazione long range
- **SIM800L/SIM7600**: GSM/4G
- **GPS Neo-6M/8M**: posizionamento

### Storage
- **SD Card**: storage principale
- **USB SSD**: storage più veloce e affidabile
- **NVMe SSD** (Pi 5): velocità massima via PCIe

### Display
- **OLED SSD1306**: 128x64, I²C, monocromatico
- **TFT ILI9341**: 320x240 colori, SPI
- **E-Ink**: basso consumo, leggibile alla luce solare
- **HDMI**: monitor standard

### Audio
- **Jack 3.5mm**: uscita audio analogica
- **USB Audio**: qualità migliore
- **I²S DAC**: audio di alta qualità (PCM5102, PCM5122)
- **HDMI Audio**: passthrough audio via HDMI

## Configurazione avanzata

### Overclocking (Pi 4/5)
```bash
# /boot/config.txt
over_voltage=6
arm_freq=2000
gpu_freq=750
```

### Boot da SSD USB (Pi 4)
```bash
sudo raspi-config → Advanced → Boot Order → USB Boot
```

### Headless setup (senza monitor)
1. Crea file `ssh` vuoto nella partizione boot
2. Crea `wpa_supplicant.conf` con credenziali WiFi
3. Collegati via SSH: `ssh pi@raspberrypi.local`

### VNC Server
```bash
sudo raspi-config → Interface Options → VNC → Enable
```

### Tailscale (VPN)
```bash
curl -fsSL https://tailscale.com/install.sh | sh
sudo tailscale up
```
